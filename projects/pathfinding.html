SAT Collision Detection

Robust 2D collision detection using the Separating Axis Theorem (SAT)

Overview

This project demonstrates a real-time implementation of SAT for collision detection between convex polygons. It includes visualization tools for debugging normals, penetration depth, and contact points.

Core Concepts

  • Convex polygon projections onto axes
  • Detection of overlap intervals
  • Resolution using minimum penetration vector
  • Dynamic object support (velocity-aware)

Demo

Live debug rendering of contact normals and overlap resolution

Technologies

  • C# with custom rendering loop
  • Debug overlays for normals and contact vectors
